# ゲートイン！ アリーナ転換 TASKLIST

## 📅 2026-05-30
## 🎯 方針：「予想できるSNS」→「予想の上手さが"証明"できる実力主義のアリーナ＋検証可能なエッジ」

背骨は **検証可能なレーティング**。これを軸に UI・機能の優先順位を全部決める。
順番：レート定義 → 画面に出す(UI刷新) → オンボーディング → 整理 → 収益化。

---

## 🟢 フェーズ1：レーティング関数（背骨）

> 詳細は `rating-spec.md`。ここが全ての前提。

- [ ] `src/lib/rating/` にスコア計算・更新の純関数を実装
- [ ] `settle-race` に組込み（人・AIを同一関数で処理）
- [ ] 7年データで `b`・`scale` を校正（基準戦略が横ばい／好成績モデルが正）
- [ ] `pred.score` 保存、履歴からレート冪等再計算
- [ ] 確定後予想の削除・編集を禁止（検証可能性の生命線）
- [ ] 既存の人・AIの過去予想をバックフィルしてレート初期化

**完了条件**：全ユーザー・全AIにレートが付き、確定のたび人もAIも同じ式で増減する。

## 🟡 フェーズ2：土台の再配置とUI刷新 ← ★UI変更はすべてここ

レートを「画面に出す」作業。ここで見た目も情報設計も刷新する。

### 2A. デザインシステム（A配色を体系化）
- [ ] カラートークン整備：ブランド緑＝主役/GO、ゴールド＝G1の特別感に限定、キャラ色は識別用、**紫グラデ廃止**
- [ ] フォント：M PLUS Rounded 1c（数字は必要に応じ Roboto Mono）
- [ ] グレード色分け（G1橙/G2赤/G3緑のベタ塗り3連）を撤廃 → 小バッジ＋アクセント帯に
- [ ] 共通カードコンポーネント化（角丸・影・余白を統一）
- [ ] 空状態の文言・表示を前向きに（「0人参加中」「最初の予想者に」等を撤去）

### 2B. ナビゲーション & 統合アリーナ
- [ ] BottomNav を **ホーム / レース / ⊕予想(中央FAB) / アリーナ / マイページ** に
- [ ] TL と ランキングを「**アリーナ**」1タブに統合
  - [ ] 共通ヘッダ「あなたの実力」（レート/順位/今週Δ/対AIライバル）
  - [ ] 切替バー：`ランキング / フォロー中 / 発見`
  - [ ] ランキング：人とAIを同列、自分の順位ピン、期間切替（月間/連勝/累計）
  - [ ] フィード：各投稿にレート・順位・**確定結果（的中±P／単勝◯円）**を付与
  - [ ] 「発見」：高レート/トレンド予想家を提示（新規の空フィード対策）
- [ ] AIの「投票後まで非表示」ゲートを**解除**（コールドスタート対策）

### 2C. ホーム再構成
- [ ] ヒーローは**1つだけ**（直近の最重要レース）。G2/G3は控えめ2枚組へ降格
- [ ] 大会は BottomNav から外し **TOPに残置**。「あと3R予想で参加」進捗＋賞金で表示
- [ ] ランキングのカードは「アリーナ」への入口に置換
- [ ] 「今すぐ予想できる」レース導線を上部に（投票＝北極星に直結）
- [ ] AIストーリーズは維持（常に予想が出るのでフィードが空にならない資産）

### 2D. プロフィール＝実績
- [ ] `/mypage`・`/users/[uid]` の主役を**レート・回収率・確定済み予想の全ログ**へ
- [ ] フォロー＝その実績の購読、という意味づけのUI
- [ ] バッジ等は実績の副産物として縮小表示

**完了条件**：アリーナで人とAIが同じレートで並び、フィードの各投稿に実績が紐づき、ナビ・ホーム・プロフィールがレート中心に再構成されている。
