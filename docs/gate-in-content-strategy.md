## 1. 記事フォーマット設計

### 1-1. 共通テンプレート構造（SEO最適化）

すべての記事は以下の構造に統一します。MicroCMSのリッチエディタで再現可能な形式です。

```
■ title（MicroCMS: テキスト）
  → 形式: 【カテゴリ名】メインキーワード + ベネフィット or 修飾語
  → 例:「東京芝2400m完全攻略｜血統・枠順・騎手データで狙い撃つ」
  → 文字数: 28〜40文字（Google検索結果で途切れない範囲）

■ slug
  → 形式: カテゴリ略称-主題の英語（ハイフン区切り）
  → 例: course-tokyo-turf-2400

■ excerpt（MicroCMS: テキストエリア / meta description兼用）
  → 文字数: 80〜120文字
  → 構成: 課題提示 → 記事の内容 → 読むメリット
  → 例:「東京芝2400mの好走血統・有利枠・騎手成績をデータで徹底分析。
        ダービーや天皇賞（秋）の予想に直結する攻略ポイントを解説します。」

■ content（MicroCMS: リッチエディタ）構成:
  ┌─────────────────────────────────────────┐
  │ リード文（150〜200字）                      │
  │ → 読者の課題共感 + この記事で分かること      │
  ├─────────────────────────────────────────┤
  │ 📋 この記事でわかること（箇条書き3〜5項目）   │
  │ → アンカーリンクで各見出しへジャンプ          │
  ├─────────────────────────────────────────┤
  │ H2: 見出し1（メインキーワード含む）          │
  │   H3: 小見出し                              │
  │   本文 + データ表 or 図解                   │
  ├─────────────────────────────────────────┤
  │ H2: 見出し2                                │
  │   H3: 小見出し                              │
  │   本文 + 具体例（馬名・レース名）            │
  ├─────────────────────────────────────────┤
  │ H2: 見出し3                                │
  │   実践的な活用法 / まとめ                    │
  ├─────────────────────────────────────────┤
  │ 🎯 クイズに挑戦CTA（hasQuiz: true の場合）  │
  ├─────────────────────────────────────────┤
  │ 📖 関連記事（2〜3件）                       │
  └─────────────────────────────────────────┘

■ readTime → 本文文字数 ÷ 500（日本語の平均読速）で自動計算
■ tags → 記事内の固有名詞・キーワードから3〜6個
■ emoji → カテゴリに応じたアイコン（サムネイル代替）
```

### 1-2. カテゴリ別テンプレート

各カテゴリで「型」を決めておくと、執筆（AI生成含む）の品質が安定します。

## 3. クイズフォーマット設計 & サンプル問題

### 3-1. クイズ作成フォーマット

```json
{
  "question": "問題文（簡潔で明確に。40〜80字）",
  "category": "quiz-categories のID",
  "level": "beginner | intermediate | advanced | master",
  "choice1": "選択肢A",
  "choice2": "選択肢B",
  "choice3": "選択肢C（任意）",
  "choice4": "選択肢D（任意）",
  "correctIndex": 1,
  "explanation": "<p>正解は<strong>選択肢A</strong>です。</p><p>解説本文（100〜200字）。なぜその答えなのかの根拠 + 関連知識を1つ追加。</p>",
  "order": 1
}
```

**難易度ガイドライン:**

| レベル | 正答率目標 | 出題傾向 |
|--------|-----------|---------|
| beginner（入門） | 70〜90% | 有名な馬名・基本用語・常識レベル |
| intermediate（中級） | 40〜70% | 具体的なデータ・やや細かい知識 |
| advanced（上級） | 20〜40% | 数字の正確さ・歴史的事実の詳細 |
| master（マスター） | 5〜20% | マニアックな知識・引っかけ要素あり |

**解説の書き方ルール:**
1. 最初の一文で正解を明示する
2. 「なぜその答えか」の根拠を示す（データ・事実）
3. 不正解の選択肢についても簡単に触れる（「Bの〇〇はG1 5勝」など）
4. 関連する記事へのリンクを示唆する文を入れる
5. 150字以内でコンパクトに

#### 🧬 血統マスター

**Q1 [beginner]**
- 問題: ディープインパクトの父はどの馬？
- ① サンデーサイレンス ② ノーザンテースト ③ ブライアンズタイム ④ トニービン
- 正解: ① サンデーサイレンス
- 解説: ディープインパクトの父はサンデーサイレンスです。サンデーサイレンスは日本競馬史上最大の影響力を持つ種牡馬で、リーディングサイアーを13回獲得しました。

**Q2 [intermediate]**
- 問題: ロードカナロア産駒が最も高い勝率を記録している距離帯は？
- ① 1000〜1200m ② 1400〜1600m ③ 1800〜2000m ④ 2200m以上
- 正解: ② 1400〜1600m
- 解説: ロードカナロアはスプリンター（短距離馬）のイメージがありますが、産駒はマイル前後の1400〜1600mで最も安定した成績を残しています。アーモンドアイに代表されるように中距離でも活躍する馬が出ています。

**Q3 [intermediate]**
- 問題: 「母父」のことを専門用語で何と呼ぶ？
- ① サイアー ② ブルードメアサイアー ③ ダムサイアー ④ ②と③の両方
- 正解: ④ ②と③の両方
- 解説: 母父は「ブルードメアサイアー（BMS）」とも「ダムサイアー」とも呼ばれます。産駒の能力に大きな影響を与える要素で、血統分析では父と同じくらい重要視されます。

**Q4 [advanced]**
- 問題: ディープインパクト産駒のG1勝利数が最も多い馬は？
- ① ジェンティルドンナ ② コントレイル ③ グランアレグリア ④ アーモンドアイ
- 正解: ① ジェンティルドンナ
- 解説: ジェンティルドンナはG1通算7勝で、ディープインパクト産駒として最多記録です。コントレイルは5勝、グランアレグリアは6勝。なおアーモンドアイ（G1 9勝）はロードカナロア産駒なので注意。

**Q5 [master]**
- 問題: サンデーサイレンス系で日本のリーディングサイアーを獲得していない種牡馬はどれ？
- ① ディープインパクト ② キングカメハメハ ③ ハーツクライ ④ ステイゴールド
- 正解: ② キングカメハメハ
- 解説: キングカメハメハはキングマンボ系であり、サンデーサイレンス系ではありません。ミスターシービーの血を引くキングマンボ系に属します。他の3頭はいずれもサンデーサイレンスの直系です。

#### 🏇 G1メモリアル

**Q1 [beginner]**
- 問題: 日本ダービー（東京優駿）が行われる競馬場はどこ？
- ① 中山競馬場 ② 東京競馬場 ③ 阪神競馬場 ④ 京都競馬場
- 正解: ② 東京競馬場
- 解説: 日本ダービーは毎年5月末〜6月初旬に東京競馬場の芝2400mで行われます。3歳馬の頂点を決める「競馬の祭典」と呼ばれています。

**Q2 [beginner]**
- 問題: 牡馬クラシック三冠レースに含まれないのはどれ？
- ① 皐月賞 ② 日本ダービー ③ 天皇賞（秋） ④ 菊花賞
- 正解: ③ 天皇賞（秋）
- 解説: 牡馬クラシック三冠は皐月賞（中山2000m）、日本ダービー（東京2400m）、菊花賞（京都3000m）の3レースです。天皇賞（秋）は古馬も出走可能な別カテゴリのG1です。

**Q3 [intermediate]**
- 問題: 有馬記念がファン投票で出走馬を選ぶ方式なのはいつから？
- ① 第1回から ② 1960年代から ③ 1970年代から ④ 1980年代から
- 正解: ① 第1回から
- 解説: 有馬記念は1956年の第1回から一貫してファン投票で出走馬が選ばれています。正式名称は「有馬記念（グランプリ）」で、中山競馬場の芝2500mで行われる年末の風物詩です。

**Q4 [advanced]**
- 問題: ジャパンカップで初めて外国馬が優勝した年は？
- ① 1981年（第1回） ② 1982年 ③ 1983年 ④ 1985年
- 正解: ① 1981年（第1回）
- 解説: 1981年の第1回ジャパンカップはアメリカのメアジードーツが優勝しました。創設当初は外国馬が上位を独占する時代が続き、日本馬の初勝利は1984年のカツラギエースでした。

**Q5 [master]**
- 問題: JRA芝G1の1レースにおける最大着差（レコード）に最も近いのは？
- ① 5馬身 ② 9馬身 ③ 11馬身 ④ 15馬身
- 正解: ③ 11馬身
- 解説: 2020年の天皇賞（秋）でアーモンドアイとの接戦をフィエールマンが繰り広げた一方、菊花賞でのナリタブライアンの7馬身差やディープインパクトの菊花賞など大差勝ちは複数ありますが、圧倒的な着差として記録に残るレースは複数存在します。

#### 👨‍✈️ 騎手検定

**Q1 [beginner]**
- 問題: 武豊騎手が達成したJRA通算勝利数のマイルストーンとして正しいのは？
- ① 3000勝 ② 4000勝 ③ 4400勝以上 ④ 5000勝
- 正解: ③ 4400勝以上
- 解説: 武豊騎手はJRA通算4400勝以上を達成しており、日本競馬史上最多勝騎手です。2023年にも記録を更新し続けています。

**Q2 [beginner]**
- 問題: 2023〜2024年にJRAリーディングジョッキーを獲得した騎手は？
- ① C.ルメール ② 川田将雅 ③ 横山武史 ④ 戸崎圭太
- 正解: ① C.ルメール
- 解説: クリストフ・ルメール騎手は近年のJRAリーディングジョッキーの常連です。安定した騎乗技術と高い勝率で日本競馬のトップに君臨しています。

**Q3 [intermediate]**
- 問題: 「短期免許」で来日する外国人騎手の免許期間は最長何か月？
- ① 1か月 ② 3か月 ③ 6か月 ④ 1年
- 正解: ② 3か月
- 解説: 短期免許は最長3か月間です。毎年春・秋のG1シーズンに合わせて有力騎手が来日します。通年免許を持つルメール騎手やC.デムーロ騎手とは異なる制度です。

**Q4 [advanced]**
- 問題: 日本ダービーの最多騎乗回数（2024年時点）を持つ現役騎手は？
- ① 武豊 ② 横山典弘 ③ 蛯名正義（引退） ④ 柴田善臣
- 正解: ① 武豊
- 解説: 武豊騎手はダービー騎乗回数が30回を超え、そのうち6勝を挙げています。ダービージョッキーとして最も多くの栄冠に輝いた騎手です。

**Q5 [master]**
- 問題: 2024年のJRA全体の騎手勝率（全騎手平均）に最も近いのは？
- ① 約5% ② 約7% ③ 約10% ④ 約13%
- 正解: ② 約7%
- 解説: JRA全体の平均勝率はおよそ6〜8%前後です。リーディング上位の騎手は15〜20%の勝率を記録することもあり、平均との乖離が騎手の実力差を物語っています。

#### 🏟️ コース攻略

**Q1 [beginner]**
- 問題: JRAの競馬場で最も直線が長いのはどこ？
- ① 東京競馬場 ② 阪神競馬場 ③ 中山競馬場 ④ 新潟競馬場（外回り）
- 正解: ④ 新潟競馬場（外回り）
- 解説: 新潟競馬場の外回りコースは直線が約659mあり、JRA全場で最長です。東京競馬場（約526m）も長いですが、新潟には及びません。直線が長いコースでは差し・追い込み馬が有利になりやすいです。

**Q2 [beginner]**
- 問題: 中山競馬場の芝コースの特徴として最も正しいのは？
- ① 平坦でスピード向き ② 急坂があり小回り ③ 直線が600m以上 ④ 左回り
- 正解: ② 急坂があり小回り
- 解説: 中山競馬場は右回りで、ゴール前に急な上り坂があるのが特徴です。小回りコースのため先行馬が有利になりやすく、パワーとスタミナが要求されます。

**Q3 [intermediate]**
- 問題: 東京芝2400m（日本ダービーのコース）のスタート地点はどこ？
- ① 正面スタンド前 ② 向こう正面 ③ 1コーナー付近 ④ 2コーナー奥のポケット
- 正解: ④ 2コーナー奥のポケット
- 解説: 東京芝2400mは2コーナー奥のポケット地点からスタートし、向こう正面→3コーナー→4コーナー→長い直線という流れです。最初のコーナーまでの距離が長いため、枠順の有利不利は比較的少ないとされています。

**Q4 [advanced]**
- 問題: 阪神芝コースの「内回り」と「外回り」で直線距離の差はおよそ何m？
- ① 約60m ② 約120m ③ 約180m ④ 約240m
- 正解: ② 約120m
- 解説: 阪神の内回りは直線約356m、外回りは約474mで、差は約120mです。同じ阪神でも内回りは先行有利、外回りは差し有利と傾向が大きく異なるため、コース選択の確認は予想の基本です。

**Q5 [master]**
- 問題: JRA10場のうち「左回り」の競馬場はいくつある？
- ① 2場 ② 3場 ③ 4場 ④ 5場
- 正解: ② 3場
- 解説: 左回りの競馬場は東京・中京・新潟の3場です。残りの札幌・函館・福島・中山・阪神・京都・小倉はすべて右回りです。左回りが得意な馬を見極めることも馬券力の一つです。

## 4. コンテンツ大量生成戦略

### 4-1. AI生成フロー

```
STEP 1: テンプレート準備
  → 上記カテゴリ別テンプレートをプロンプトとして用意
  → 各テンプレートに「必須要素」「文体ルール」「文字数目安」を明記

STEP 2: Claude APIによるバッチ生成
  → プロンプト例（記事）:
    「あなたは競馬ライターです。以下のテンプレートに従って記事を作成してください。
     カテゴリ: 血統
     トピック: キタサンブラック産駒の特徴と狙い方
     テンプレート: [上記テンプレート構造]
     出力形式: MicroCMSのリッチエディタ用HTML
     注意: データは最新のものを使用。推測は明示すること。」

  → プロンプト例（クイズ）:
    「以下の形式で競馬クイズを10問作成してください。
     カテゴリ: 血統マスター
     難易度: intermediate
     出力形式: JSON配列
     制約:
      - 選択肢は4つ（うち正解は1つ）
      - 解説は150字以内
      - 具体的なデータ・事実に基づくこと
      - 紛らわしい選択肢を含めて難易度を調整」

STEP 3: レビュー & ファクトチェック
  → 生成された記事のデータ（成績・数字）をJRA公式等で検証
  → クイズの正解が本当に正しいか確認
  → 文体・トーンの統一感をチェック

STEP 4: MicroCMS投入
  → MicroCMS Management APIを使って一括登録
```

### 4-2. 品質管理プロセス

```
■ ファクトチェックリスト
  □ 馬名の漢字・カタカナ表記が正確か
  □ G1勝利数・通算成績の数字が正しいか
  □ コースの距離・形態の記述が正確か
  □ 騎手・調教師の所属・成績が最新か
  □ クイズの正解が事実に基づいているか

■ SEOチェックリスト
  □ タイトルに主要キーワードが含まれているか（28〜40字）
  □ excerptがmeta descriptionとして適切か（80〜120字）
  □ H2見出しにキーワードが自然に入っているか
  □ 内部リンク（関連記事）が設定されているか
  □ slugが英字ハイフン区切りで適切か

■ UX品質チェックリスト
  □ readTimeが本文量に対して適切か
  □ emoji（サムネイル代替）がカテゴリと合致しているか
  □ tagsが3〜6個設定されているか
  □ hasQuizが正しく設定されているか
  □ クイズの解説に「関連記事を読む」誘導が入っているか
```

### 4-3. MicroCMS一括投入方法

MicroCMS Management APIを使ったNode.jsスクリプトで一括登録します。

```typescript
// 概要フロー
// 1. JSON/CSVでコンテンツデータを準備
// 2. MicroCMS Management API（POST /api/v1/{endpoint}）でcreate
// 3. カテゴリ・タグは先にIDを取得しておく
// 4. 画像がある場合はMedia APIで先にアップロード

// エンドポイント
// 記事: POST https://gatein.microcms.io/api/v1/articles
// クイズ: POST https://gatein.microcms.io/api/v1/quiz-questions

// ヘッダー
// X-MICROCMS-API-KEY: {WRITE用APIキー}
// Content-Type: application/json

// 推奨バッチサイズ: 1リクエスト/秒（APIレート制限考慮）
// エラーハンドリング: 失敗分をログに記録してリトライ
```

### ロングテール狙いの理由

「競馬 予想」のようなビッグキーワードはnetkeiba・JRA等の大手と競合して厳しいため、ゲートイン！では以下のようなロングテールKWを中心に攻めるのが効果的です。

- 「{種牡馬名} 産駒 特徴」（例: キタサンブラック 産駒 特徴）
- 「{競馬場} {距離} 攻略」（例: 東京 芝2400m 攻略）
- 「{騎手名} 得意コース」（例: ルメール 得意コース）
- 「競馬 {予想手法} やり方」（例: 競馬 展開予想 やり方）
- 「{馬名} 産駒 / 血統 / 伝説」（例: オグリキャップ 伝説）
