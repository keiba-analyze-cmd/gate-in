# ゲートイン！ デザインシステム仕様 (design-system)

## 📅 2026-05-30 ／ フェーズ2A

確認用: `gatein-styleguide.html`（描画版）／ 実装: `src/app/theme.css`（Tailwind v4トークン）

---

## 🎯 原則

各ページの刷新を「描き直し」でなく「**部品の差し替え**」にするための土台。
トークン＋共通コンポーネントを1回作れば、全ページがほぼ自動で揃う。

1. **色は意味で割り当てる。** 緑＝主役/GO/アクティブ、ゴールド＝**G1・賞金のみ**、赤＝締切/危険/外し。装飾目的で色を増やさない。
2. **ヒーローは1画面に1つ**（直近の最重要レース）。グレードの全面ベタ塗り3連は廃止。
3. **キャラ識別色はアバター/ドット専用**（ボタン・見出しには使わない）。
4. **数値はモノスペース**（レート/オッズ/締切/回収率 = Roboto Mono）。
5. **紫グラデは使わない**（大会は紫→ゴールド）。
6. **空状態は前向きに**（「0人参加中」「最初の予想者に」を撤去）。
7. **カードは共通 `.card`**。角丸・境界・影を統一し、ページごとに作り直さない。

|------|------|--------|--------|

| 境界 | `--line` | #efe7da | #352c22 |
| 主役/GO | `--brand` / `--brand-strong` / `--brand-soft` | #159a57 / #0f7e46 / #e2f3ea | #2bb673 / #1f9d60 / #15301f |
| G1/賞金 | `--gold` / `--gold-strong` / `--gold-soft` | #caa24a / #8a6a17 / #f6edd6 | #e0c074 / #e0c074 / #2e2616 |
| 締切/危険/外し | `--danger` / `--danger-soft` | #d63333 / #ffe6e6 | #ef6a6a / #3a1f1f |
| ○など | `--info` / `--info-soft` | #2563c9 / #e6f0ff | #6fa0ee / #1c2740 |
| △ | `--osae` / `--osae-soft` | #b27400 / #fff2db | #d6a44a / #2e2614 |

**グレード**（小バッジ）: G1=`--g1`(金) / G2=`--g2`(赤) / G3=`--g3`(青)
**キャラ識別**: `--c-hayate`(青) `--c-kazan`(赤) `--c-hakusen`(緑) `--c-hibari`(橙) `--c-gantetsu`(灰)
**半径**: `--r-sm`8 / `--r`14 / `--r-lg`18 / `--r-xl`22
**フォント**: 見出し本文=`--font-display`(M PLUS Rounded 1c) / 数値=`--font-num`(Roboto Mono)

> ダークは `:root` と `.dark` で同名変数を切替。`@theme inline` でユーティリティ化（`bg-surface` `text-ink` `font-display` 等が自動で両対応）。

## 🌙 ダーク対応

トークンで両対応済み。注意点だけ:
- ブランド緑は**両テーマで主役**（識別の一貫性）。ゴールドが特別感アクセント。
- 大会のゴールドグラデは、ダークでは暗いゴールド面 + ゴールド文字に切替（`.dark .contest`）。
- ヒーローの緑グラデは共通。影だけダークで強める。

## 🗺 ページ別の適用方針

| ページ | 対応 | 内容 |
|--------|------|------|
| ホーム | 済 | リデザイン反映 |

| アリーナ（新） | **Tier1** | 部品をそのまま組む |
| プロフィール | **Tier1** | レート・回収率・全予想ログを主役に |
| LP | **Tier1** | 新ブランド＋「検証可能なエッジ」の訴求に |
| 大会 | Tier2 | 紫廃止・大会カード部品へ |
| AI予想家ページ | Tier2 | レート表示を追加 |
| 通知・検索 | Tier2 | 軽い化粧直し |
| **道場系** | **触らない** | フェーズ4で停止/転用予定 |
| **マイページ副ページ** | **触らない** | レートへ統合予定のものは刷新しない |
| 規約・特商法・admin | 最小 | 衝突しなければ放置 |
